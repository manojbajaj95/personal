# Opinionated Tools Directory

List of opinionated tools to launch your product faster. Most of the tools have generous free tier unless mentioned specifically. All the tools are tested personally and preferred by me.

- Wireframing
  [Relume AI Site Builder](https://library.relume.io/ai-site-builder) & [Relume Ipsum](https://www.relumeipsum.com/)
  [Figma](https://www.notion.so/Opinionated-Tools-Directory-c1f3fa3d75514d8fa181055984d4e80b?pvs=21): [https://www.figma.com/blog/how-to-wireframe/](https://www.figma.com/blog/how-to-wireframe/)
- Website Inspiration
  [Dribble](https://dribbble.com/search)
- Theming & Fonts
  [https://www.realtimecolors.com/](https://www.realtimecolors.com/)
  [https://aicolors.co/](https://aicolors.co/)
  [https://fonts.google.com/](https://fonts.google.com/)
- Logo
  [https://www.namecheap.com/logo-maker/](https://www.namecheap.com/logo-maker/)
  [Vector Art](https://vectorart.ai/) to convert to SVG if needed
- Images / illustrations
  [https://unsplash.com/](https://unsplash.com/)
  [Undraw](https://undraw.co/illustrations) for Illustrations
- Building Landing Page
  - Code Based Solution
    Frontend: [NextJS](https://nextjs.org/) / [https://tx.shadcn.com/](https://tx.shadcn.com/) for Starter Kit
    [v0](https://v0.dev/) for AI generated pages/components
    Components: [Shadcn UI](https://ui.shadcn.com/) , [Aceternity](https://ui.aceternity.com/components)
    Icons: [Lucide](https://lucide.dev/icons/)
    Theme + Font : [Realtime Colors](https://realtimecolors.com/)
    Hosting: [Vercel](https://vercel.com/)
    CMS: Markdown in next js available in starter kit
  - No Code
    - Framer
    - Webflow
  - Waitlist: Embed forms [Tally.so](https://tally.so/)
  - Contact Form: Embed [Contact Form](https://tally.so/templates/contact-form/dnWe3O)
  - Newsletter: [Substack](https://substack.com/)
- Building MVP
  - Code based solution
    Authentication: [Supabase](https://supabase.com/) for Auth & DB
    - Database:
      - SQL: Supabase
      - NoSQL: MongoDB Atlas
      - Redis: Upstash
        Frontend: NextJS
        AI: Mixtral 8x7 on [TogetherAI](https://www.together.ai/) , free 25$ credits
        Backend: NextJS if Javascript Dev / FastApi if python
        Hosting: [Vercel](https://vercel.com/)
